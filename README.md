# Roam. 🌿

> Track events together. Watch your shared garden grow.

A free, browser-based event calendar for up to 4 friends. Every event you attend plants a flower in your shared garden.

## Features
- 📅 Shared weekly/monthly calendar for up to 4 people
- 🌐 Paste any event URL to auto-fill details
- 🌸 Animated garden that grows as you attend events
- 📷 Add photos to events
- 🔔 Upcoming event reminders
- 📲 Installable as a home screen app (PWA)
- 💾 All data stored locally — no server, no accounts required beyond signup

## Deploy to GitHub Pages (free, 5 minutes)

1. Fork or upload these files to a new GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Click **Save**
5. Your app is live at `https://YOUR-USERNAME.github.io/REPO-NAME/bloom-calendar.html`

## Files

| File | Purpose |
|------|---------|
| `bloom-calendar.html` | The entire app — HTML, CSS, JS in one file |
| `manifest.json` | PWA manifest — enables "Add to home screen" |
| `sw.js` | Service worker — offline support and caching |
| `icon.svg` | App icon source |

## Install as a phone app

**iPhone/iPad (Safari):**
1. Open the app URL in Safari
2. Tap the Share button (box with arrow)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" — the Roam. icon appears on your home screen

**Android (Chrome):**
1. Open the app URL in Chrome
2. A banner will appear automatically offering to install
3. Or tap the three-dot menu → "Add to Home Screen"
4. The app installs and works like a native app
